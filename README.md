# geoip resolver
Simple service to serve GEOIP data based on GEOLITE by MAXMIND

## Instalation

```bash
$ npm ci
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# tests coverage
$ npm run test:cov
```
